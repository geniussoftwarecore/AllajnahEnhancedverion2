# دليل اختبار ميزة التصنيفات حسب الجهة

## خطوات الاختبار:

### 1. تسجيل الدخول
- انتقل إلى `/login`
- سجل دخول بأحد الحسابات التالية:
  - **حساب تاجر**: `essam.mahdi@gmail.com` (إذا كانت كلمة المرور معروفة)
  - **أو**: سجل دخول كـ **لجنة عليا** (`taha.kohlani@gmail.com`) وأنشئ حساب تاجر جديد

### 2. إنشاء شكوى جديدة
- بعد تسجيل الدخول كتاجر، انتقل إلى `/complaints/new`
- سترى نموذج تقديم الشكوى

### 3. اختبار الميزة
1. **اختر الجهة** من القائمة الأولى:
   - الجمارك (23 تصنيف)
   - الموصفات والمقاييس (15 تصنيف)
   - الضرائب (9 تصنيف)
   - صندوق النظافة والتحسين (9 تصنيف)

2. **بعد اختيار الجهة**، ستظهر التصنيفات الخاصة بتلك الجهة فقط في القائمة الثانية

### 4. مثال عملي
- عند اختيار **"الجمارك"**، ستظهر تصنيفات مثل:
  - اهمال البضائع عند المطابقة
  - غرامات غير قانونية
  - معاملة المواد الخام كمنتج نهائي للاستهلاك
  - ... (23 تصنيف إجمالاً)

- عند اختيار **"الموصفات والمقاييس"**، ستظهر تصنيفات مثل:
  - سحب عينات متكررة لنفس المنتج
  - عدم قبولهم الفحوصات السابقة
  - مبالغة في الاجراءات الشكلية والمعاينة الظاهرية
  - ... (15 تصنيف إجمالاً)

## التحقق من البيانات (للمطورين)

### API Endpoints:
```bash
# الحصول على جميع الجهات
curl http://localhost:8000/api/government-entities

# الحصول على جميع التصنيفات
curl http://localhost:8000/api/categories

# الحصول على تصنيفات جهة معينة
curl 'http://localhost:8000/api/categories?government_entity=الجمارك'
```

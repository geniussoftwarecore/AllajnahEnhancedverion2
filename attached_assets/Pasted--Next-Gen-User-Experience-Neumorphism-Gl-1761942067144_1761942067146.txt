🎨 الرؤية الجديدة للواجهة (Next-Gen User Experience)
💫 النمط العام

أسلوب التصميم: Neumorphism + Glassmorphism (مظهر زجاجي ناعم بعمق خفيف)

الألوان:

الأخضر الرئيسي: #00C46B

تدرج مائل للأخضر الفاتح حتى الأبيض: linear-gradient(135deg, #00C46B 0%, #CFFFE0 100%)

الخلفية العامة: أبيض ناعم جدًا مع ظل أخضر شبه شفاف

الخطوط:

عربية: Cairo أو Tajawal

إنجليزية: Inter

حجم خط متناسق مع فراغات واسعة ومسافات هواء (White Space)

🧭 التنقل الذكي (Smart Navigation)

الشريط الجانبي: أيقونات كبيرة متحركة عند المرور (Hover expand animation).

العنوان العلوي: شفاف قليلاً (Glass Header) مع شريط بحث + إشعارات + بروفايل.

التنقل بين الصفحات: انتقال ناعم (Framer Motion — Slide & Fade).

دعم الوضع الليلي (Dark Mode) بضغطة واحدة.

🤝 التفاعل الذكي (Smart Interaction)

البطاقات تتنفس: تتحرك تلقائيًا عند المرور + إضاءة جانبية باللون الأخضر.

إيماءات المستخدم:

سحب من اليمين → فتح تفاصيل الشكوى.

نقر مزدوج → فتح نافذة تعديل.

سحب للأسفل في الهاتف → تحديث القائمة (Smart Pull-to-Refresh).

رسائل النظام:

Toast Notifications مع أيقونات متحركة (Lottie Animations).

📱 العرض المتجاوب (Adaptive Layout)

شاشات كبيرة: Dashboard بثلاث أعمدة.

الأجهزة اللوحية: عمودان فقط + شريط علوي قابل للطي.

الهواتف: قائمة واحدة + Bottom Navigation بأيقونات متحركة.

🧩 مثال على كود تصميم حديث (React + Tailwind + Framer Motion)
import { motion } from "framer-motion";

export default function ComplaintCard({ title, status, date }) {
  const colors = {
    open: "from-green-400 to-green-200",
    pending: "from-yellow-400 to-yellow-200",
    closed: "from-gray-300 to-gray-100",
  };

  return (
    <motion.div
      whileHover={{ scale: 1.03, boxShadow: "0 10px 25px rgba(0, 196, 107, 0.2)" }}
      className={`p-6 rounded-3xl bg-gradient-to-br ${colors[status]} backdrop-blur-lg border border-white/40 transition-all`}
    >
      <h3 className="text-gray-900 font-semibold text-lg">{title}</h3>
      <p className="text-sm text-gray-600 mt-1">{date}</p>
      <span className="mt-3 inline-block bg-white/30 px-3 py-1 rounded-full text-sm font-medium">
        {status.toUpperCase()}
      </span>
    </motion.div>
  );
}

🚀 الميزات الحديثة التي سيضيفها التصميم الجديد
الميزة	الوصف
Dynamic Gradient Backgrounds	خلفيات متغيرة بتدرجات الأخضر المتحركة.
Smart Animations	استخدام Framer Motion لكل انتقال.
Glassmorphism Panels	نوافذ نصف شفافة أنيقة بحدود ناعمة.
Gesture UX	تفاعل بالسحب واللمس للهواتف.
Modern Dashboard Layout	واجهة إدارية جديدة مستوحاة من لوحات إدارة SaaS العالمية.
Micro-Interactions	كل زر واستجابة تملك حركة صغيرة (Ripple, Bounce, Fade).
